# crypto-tracker

A responsive cryptocurrency tracking web application built using HTML, CSS, and JavaScript. The application fetches real-time cryptocurrency data using an API and allows users to explore cryptocurrency prices, market information, and manage their personal crypto portfolio.

## Features

- View real-time cryptocurrency prices
- Display cryptocurrency market information
- Search and explore different cryptocurrencies
- Track cryptocurrencies through a portfolio page
- Dynamic data fetching using API integration
- Separate portfolio management functionality
- Responsive design for desktop and mobile screens
- Interactive user interface
- Interactive charts and visualization of cryptocurrency

## Technologies Used

- HTML5
- CSS3
- JavaScript
- CoinGecko API

## Project Overview

The Crypto Tracker application provides users with updated cryptocurrency information through API integration. The application fetches cryptocurrency data and dynamically displays details such as coin prices and market information with charts and different time ranges for each.

The project also includes a portfolio section where users can manage and track selected cryptocurrencies.It will also indicate the user if he/she is in profit or loss.

## Project Structure

Crypto-Tracker/
│
├── index.html # Main cryptocurrency tracking page
├── portfolio.html # Portfolio management page
├── style.css # Application styling
├── script.js # Cryptocurrency data fetching and display logic
└── portfolio.js # Portfolio functionality

## How It Works

1. The application sends requests to the CoinGecko API using JavaScript.
2. Cryptocurrency data is fetched and processed.
3. The information is dynamically displayed on the webpage.
4. Users can navigate to the portfolio section to track selected cryptocurrencies.

## API Used

CoinGecko API

The application uses CoinGecko API to retrieve cryptocurrency market data.

## Installation and Setup

Follow these steps to run the project locally:

1. Clone the repository:

git clone https://github.com/ana0020/crypto_project.git

2. Navigate to the project folder.

3. Open `index.html` in your browser.

No additional installation or dependencies are required.
